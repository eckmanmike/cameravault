Cloud Sync Setup
Connect to Firebase
Firebase is Google's free cloud database. Once connected, your PC web app and this phone app will share the same live database — changes on one appear instantly on the other.
1
Create a free Firebase project
Go to console.firebase.google.com, sign in with Google, click Add project, name it "CameraVault", and follow the prompts. Disable Google Analytics when asked.
2
Enable Realtime Database
In the left sidebar click Build → Realtime Database → Create Database. Choose your nearest region. When asked about rules, choose Start in test mode (you can lock it down later).
3
Get your Web App config
Go to Project Settings → Your apps → Add app → Web (</>). Register the app and copy the firebaseConfig object shown. You need the values below.
4
Paste your config values
Fill in the fields below and tap Connect.
